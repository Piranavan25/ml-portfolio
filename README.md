# AI/ML Portfolio – Piran Sana

Welcome to my AI/ML project portfolio. This repository showcases a variety of machine learning and computer vision projects, including natural language processing, image classification, and real-time hand gesture recognition.

Each project demonstrates applied ML/AI skills, coding best practices, and hands-on experimentation.

---

## Projects

### 1. Tokenizer Training Project
- **Description:** Built a Byte-Pair Encoding (BPE) tokenizer from scratch using real-world datasets. Demonstrated tokenization, encoding/decoding, and vocabulary creation.
- **Technologies:** Python, Tokenizers, NLP, Byte-Pair Encoding
- **Folder:** [tokenizer-project](./tokenizer-project)
- **Highlights:** 
  - Custom tokenizer trained on real dataset
  - Supports special tokens
  - Encoding and decoding examples

### 2. Fashion MNIST Classifier
- **Description:** Convolutional Neural Network to classify Fashion MNIST images into 10 categories. Applied data augmentation, batch normalization, dropout, and visualized training metrics.
- **Technologies:** Python, TensorFlow, Keras, NumPy, Matplotlib
- **Folder:** [fashion-mnist-classifier](./fashion-mnist-classifier)
- **Highlights:** 
  - Deep CNN with regularization techniques
  - Training/validation curves and confusion matrix
  - Sample predictions visualized
  - Optional: deployable demo

---

### 3. Hand Gesture Recognition
- **Description:** Real-time hand gesture recognition system using MediaPipe and OpenCV. Implemented ML-based classification of hand landmarks with frame smoothing and gesture labeling.
- **Technologies:** Python, OpenCV, MediaPipe, Scikit-Learn, NumPy
- **Folder:** [hand-gesture-recognition](./hand-gesture-recognition)
- **Highlights:**
  - Real-time webcam detection
  - ML classifier trained on hand landmarks
  - Smoothing over frames for stable predictions
  - Evaluation-ready with confusion matrix

---

### 4. ML-Based Rock–Paper–Scissors Game
- **Description:** A machine learning–powered Rock–Paper–Scissors game that uses MediaPipe for hand landmark extraction and a custom-trained ML model to classify user gestures in real time. Integrated with a simple game engine that lets users play against the computer.
- **Technologies:** Python, OpenCV, MediaPipe, Scikit-Learn, NumPy, Joblib
- **Folder:** [rps-ml-game](./rps-ml-game)
- **Highlights:**
  - Real-time gesture classification for Rock, Paper, Scissors
  - Custom ML model trained using hand landmarks
  - Interactive gameplay with score tracking
  - Frame smoothing to reduce prediction noise
