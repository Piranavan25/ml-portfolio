# MNIST Digit Classifier (Neural Network)

## Description
Implemented a feedforward neural network to classify handwritten digits from the MNIST dataset. Achieved above 97% accuracy using a 3-layer dense architecture with ReLU activations and softmax output. Includes evaluation, confusion matrix, and visualization.

## Technologies
- Python
- TensorFlow / Keras
- NumPy
- Matplotlib
- Seaborn

## Features
- Data preprocessing and normalization
- Neural network with 128–64–10 architecture
- Training accuracy & loss visualization
- Confusion matrix heatmap
- Model saving in Keras/H5 format

## How to Run
Upload notebook to Google Colab and run all cells.
